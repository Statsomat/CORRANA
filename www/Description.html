Description

Upload your data as a CSV file and let the Statsomat app automatically compute for you pairwise correlations or associations between continuous variables. The Statsomat app delivers a PDF report containing the numerical results, a statistical interpretation of the results in plain English and the R code used to generate the results. Data files can be uploaded up to a size of 1000 KB and a maximum of 1000 rows.

Upload your data and download the report in seconds!
