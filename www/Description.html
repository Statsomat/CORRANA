Description

Upload your data as a CSV file and let the Statsomat app automatically compute for you pairwise correlations or associations between continuous variables. The Statsomat app delivers a PDF report containing numerical results, graphics and a statistical interpretation in plain English. Optionally, you can include the R code used to generate the results. The R code is ready to go: Just copy and paste it, run it to generate the same results on your own.

Upload your data and download the report in seconds!
