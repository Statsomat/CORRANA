Description

Pairwise correlation & association analysis for continuous variables with the R programming language. R Code to copy & paste & learn included. Currently, up to 1000 rows and 10 columns (e.g. variables) are permitted per dataset and app call.

Main assumptions: Continuous, independent and identically distributed random variables in pairs, only complete pairs treated (missing values are omitted), sample size greater or equal 7.
