Instructions

The data must be prepared in a certain way before it is uploaded to the Statsomat app: Each row represents a record respectively an observation. Each column represents a variable respectively a parameter of interest. Do not store multiple variables in one column (for example sex and age in one column). Apply distinct names for the columns. Empty cells indicate missing values. Other character values (e.g. "NA") are not recognized as missing values. Please take a look at these prepared dataset examples, they can help you with the data preparation.

Once you have your data prepared in the specified way, please save it with the text CSV format. If your data file is in an Excel or LibreOffice format, you can try to convert it to CSV by selecting "Save As". Try to select the format CSV with the character encoding UFT-8, this helps reading the data correctly. You can also prepare or inspect the CSV file with a text editor like Notepad++.

When your data is ready, you can proceed with the upload. A couple of upload properties must be specified. If you are not sure about them, you can choose the option "Auto" for each property. In that case, the Statsomat app will try to identify automatically the right settings. In most of the cases, this will succeed. Otherwise, please consider alternative settings or data formats.

After a successful upload of the data, you can proceed with the selection of the columns which should be considered in this app. For sample sizes smaller or equal than 5000 rows, you can select a maximum of 25 columns in each app call. For larger sample size reduce the number of selected coumns in an app call. Now you can generate the report. After a successful generation of the report, you can proceed with the download the report.

To cancel the report generation and start all over again, please reload the app from the button in the upper right corner of the screen.
